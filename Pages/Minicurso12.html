  • Minicurso: "Utilizando ferramentas de IA para organização de referências em Revisão Sistemática da Literatura"
  • Descrição:
  • Neste minicurso, temos como objetivo apresentar ferramentas que auxiliem os participantes na busca e organização do banco de dados bibliográficos para compor o acervo de revisão sistemática da literatura. Para isso, serão apresentadas estratégias de busca e seleção de material utilizando ferramentas de inteligência artificial, como também, dois organizadores de bibliografia: o Research Rabbit e o Zotero. A partir desse minicurso espera-se que os participantes possam otimizar o processo de categorização de dados de uma revisão, a partir de uma simulação simples desse trabalho.
